assumptions:

-soldier moves either y or x direction but not diagonally
-soldier can go through border and come out from opposite side


how to run:

-run all the nodes (9) as java applications
-All nodes have unique id 1 to 9
-nodes will run on port 5001 to 5009
- run the interface as java application
-give inputs for matrix , time, commander in the interface console
-give inputs for each missile


how to read output:

-matrix with positions of node
-status of each node is displayed
